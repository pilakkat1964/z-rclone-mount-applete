# Rclone Mount Management Tray Applete

A GTK4-based system tray application for managing rclone mounts with an intuitive graphical interface. This applet provides seamless integration with your desktop environment, allowing you to mount, unmount, and monitor rclone storage locations directly from your system tray.

## Key Features

- **GTK4 Interface** - Modern, responsive graphical user interface built with GTK4
- **System Tray Integration** - Quick access from your desktop system tray
- **Rclone Mount Management** - Mount and unmount remote storage locations with a single click
- **Live Mount Monitoring** - Real-time visibility into active mounts and their status
- **Rust-Based** - High-performance, memory-safe implementation in 100% Rust
